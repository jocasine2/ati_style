# AtiStyle

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ati_style', git: 'git@git.seplan.to.gov.br:jakson.moura/atistyle.git'
```

And then execute:

    $ bundle

Rails generator
    $ rails g ati_style:install
