# AtiStyle

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'AtiStyle'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install AtiStyle

Rails generator
    $ rails g ati_style:install
